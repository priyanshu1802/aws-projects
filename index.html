Project Overview

This project demonstrates a production-pattern multi-tier architecture built on AWS but deployed in Free Tier for demo.

  • Frontend via CloudFront + S3
  • EC2 behind ALB with health checks
  • RDS (MySQL) in private subnet
  • CloudFormation IaC templates
  • CloudWatch monitoring
